计算MATALB中变量的体积

Matlab中在保存数据前最好先估计一下变量的体积，但是whos命令下显示的Byte是在是让人没有直观的认识。于是，我就编写了这个函数来计算变量的体积， 然后在决定哪些数据需要保存下来。

以下是函数代码：

function ByteSize(in, fid)
% BYTESIZE writes the memory usage of the provide variable to the given file
% identifier. Output is written to screen if fid is 1, empty or not provided.

if nargin == 1 || isempty(fid)
    fid = 1;
end

s = whos('in');
fprintf(fid,[Bytes2str(s.bytes) '\n']);
end

function str = Bytes2str(NumBytes)
% BYTES2STR Private function to take integer bytes and convert it to
% scale-appropriate size.

scale = floor(log(NumBytes)/log(1024));
switch scale
    case 0
        str = [sprintf('%.0f',NumBytes) ' b'];
    case 1
        str = [sprintf('%.2f',NumBytes/(1024)) ' kb'];
    case 2
        str = [sprintf('%.2f',NumBytes/(1024^2)) ' Mb'];
    case 3
        str = [sprintf('%.2f',NumBytes/(1024^3)) ' Gb'];
    case 4
        str = [sprintf('%.2f',NumBytes/(1024^4)) ' Tb'];
    case -inf
        % Size occasionally returned as zero (eg some Java objects).
        str = 'Not Available';
    otherwise
       str = 'Over a petabyte!!!';
end
end

保存后，我们来测试一下函数的效果：

Example 01 ： 我们生成一个 1000 by 1000 的double型随机矩阵，然后利用我们刚刚编写的函数计算这个矩阵的体积。

>> A=rand(1000,1000);
>> whos
  Name         Size                Bytes  Class     Attributes

  A         1000x1000            8000000  double              

>> ByteSize(A)
7.63 Mb
>>

Example 02: 我们测试一下Matlab的极限，生成一个 900x900x100的矩阵，看看如果存储这个矩阵需要多大的空间。

>> B = rand(900,900,100);
>> whos
  Name        Size                     Bytes  Class     Attributes

  B         900x900x100            648000000  double              

>> ByteSize(B)
617.98 Mb

Oh~ 这个矩阵太大了，如果在世纪的问题中处理这类问题的话，一定要小心了。

Example 03 我们换一个复杂的结构，结构体和元组，计算它的体积。

% structure
>> Jack.Name = 'Jack Anderson';
>> Jack.Age = 30;
>> Jack.Data = rand(200,3);
>> whos
  Name      Size            Bytes  Class     Attributes

  Jack      1x1              5362  struct              

>> ByteSize(Jack)
5.24 kb
% cell array
>> A{1}=rand(100,100);
>> A{2}=rand(100,200);
>> whos
  Name      Size             Bytes  Class    Attributes

  A         1x2             240224  cell               

>> ByteSize(A)
234.59 kb

如果在编程时，一个变量的体积过大，但是若它是稀疏的，你可以将数据转换成稀疏的形式，这样的话它所占用的内存形式就会小很多了。 Example 04 稀疏矩阵，生成一个全零的矩阵，使其中只有少量元素非零，比较它原始体积与稀疏后的体积。

>> A = 1:100; 
>> MatrixA = diag(A);
>> SparseA = sparse(MatrixA);
>> whos
  Name           Size             Bytes  Class     Attributes

  A              1x100              800  double              
  MatrixA      100x100            80000  double              
  SparseA      100x100             2408  double    sparse    
>> ByteSize(MatrixA)
78.12 kb
>> ByteSize(SparseA)
2.35 kb
